    • Creating your first webpage

    BEFORE

    Before creating a webpage, there is one important step that must take place in order to be sucessful.

    Paper and pen is all you need to complete this step.Drawing it out will reveal your vision.

    Drawing it out will truly help bring your vision to life in a beautiful way.

    INITIAL STEPS

    1. Draw it out
    2. Create a folder
    3. Open folder in VS Code
    4. Boilerplate
    5. Break down into individual sections
    6. Create each section in HTML
    7. Style each section using CSS
    8. Review your color scheme

    Note:"Periodically" save your work as you go.
